<h2>Functional description:</h2>

<ul>
    <li>Get Trending movies</li>
    <li>Get movies with high ratings</li>
    <li>API Get movies by genre</li>
    <li>Take the trailer of a movie</li>
    <li>Search movies by keywords</li>
    <li>Add API authentication </li>
    <li>Added more parameters to search for movies (genre, mediaType, language, year).</li>
</ul>
